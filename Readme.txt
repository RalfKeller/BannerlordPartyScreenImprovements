PARTY SCREEN IMPROVEMENTS - 1.1.0
This mod adds improvements to the Party Screen to save you a lot of clicking:

:: Recruit all Prisoners: ::
- Recruit all available Prisoners with one click!
- Only Recruits as many as fit in your party
- Recruits the highest level prisoners first

:: Upgrade all: ::
- Upgrades all available Troops
- If a troop has multiple possible upgrades both upgrades will be used 50/50

:: Roadmap: :: 
- Sorting Prisoners and Troops
- Better Upgrade Algorithm: Set the Ratio of Infantry / Bowmen / Cavallery / Horse Archers for your army
- Tooltips

:: INSTALLATION ::
Copy the downloaded Folder to the Modules folder of your Bannerlord installation. Don't forget to activate the mod in the launcher!
